# 프로필 사이트 디자인 스펙

**날짜:** 2026-05-20  
**파일:** `profile.html` (단일 파일, HTML + CSS + JS)

---

## 1. 개요

경기북과학고등학교 2학년 학생의 개인 프로필 사이트.  
운동·게임 취미를 가진 사용자를 위한 역동적 글래스모피즘 단일 페이지.

---

## 2. 비주얼 스타일

### 색상 팔레트
| 역할 | 값 |
|---|---|
| 배경 그라데이션 | `#1e40af → #1d4ed8 → #0ea5e9 → #38bdf8` (150deg) |
| 글래스 카드 | `rgba(255,255,255,0.15)` + `backdrop-filter: blur(20px)` |
| 글래스 카드 테두리 | `rgba(255,255,255,0.28)` |
| 텍스트 (메인) | `#ffffff` |
| 텍스트 (보조) | `rgba(255,255,255,0.65~0.75)` |
| 탭 활성 텍스트 | `#1d4ed8` |
| 강조 (사선 레이어) | `rgba(255,255,255,0.05~0.07)` |

### 배경 효과
- 애니메이션 블러 블롭 3개 (`filter: blur(70px)`, `animation: drift 8s` 무한 반복)
  - 블롭 색상: `#60a5fa`, `#a78bfa`, `#34d399`
- 사선 슬래시 레이어 2개 (`clip-path: polygon`) — 상단·하단 배치

### 폰트
- Google Fonts: `Noto Sans KR` (weight 400, 500, 700, 900)

## 3. 레이아웃 구조

```
┌──────────────────────────────────────────┐
│ HEADER (fixed, glassmorphism)            │
│  [이름]          [소개] [갤러리] [링크모음] │
├──────────────────────────────────────────┤
│                                          │
│  TAB CONTENT (페이드인 전환)               │
│                                          │
└──────────────────────────────────────────┘
```

### 헤더
- `position: sticky; top: 0; z-index: 20`
- 배경: `rgba(255,255,255,0.12)` + `backdrop-filter: blur(20px)`
- 이름(좌) + 탭(우) 구성
- 탭 컨테이너: `rgba(0,0,0,0.2)` pill 배경
- 활성 탭: `rgba(255,255,255,0.92)` 배경 + `color: #1d4ed8` + scale(1.04) 트랜지션

## 4. 섹션별 상세

### 소개 탭
- 레이아웃: 프로필 사진(좌) + 이름·학교·뱃지(우) 가로 배치
- **프로필 사진**: 100×100px 원형, 글래스 배경, `animation: pulse-ring` (링 맥박 효과)
- **이름**: `font-size: 2rem; font-weight: 900`
- **학교**: `rgba(255,255,255,0.75)` 보조 텍스트
- **뱃지**: 🏃 운동, 🎮 게임 — pill 모양 글래스 카드
- **자기소개 카드**: 가로 폭 100% (max 520px), 왼쪽 3px 흰 보더 포인트, 글래스 배경

### 갤러리 탭
- 3열 그리드, gap 0.75rem
- 카드: 1:1 비율, 글래스 배경, `border-radius: 14px`
- 호버: `transform: scale(1.05) translateY(-2px)` + 그림자 강화
- 기본 슬롯 9개 (`<img>` src 교체로 사진 등록)
- 각 카드 우상단 `::after` 광택 효과 (`linear-gradient` 오버레이)

### 링크 모음 탭
- 세로 목록, max-width 440px
- 각 항목: 아이콘(40px 둥근 사각형) + 플랫폼명 + 핸들 + 화살표
- 호버: `translateX(6px)` 슬라이드 + 배경 밝아짐 + 화살표 이동
- 기본 링크: Instagram, Twitter/X, 추가 슬롯
- 아이콘 색상:
  - Instagram: `#f43f5e → #a855f7` 그라데이션
  - Twitter/X: `#0f172a → #334155` 다크
  - 추가: `#0284c7 → #0ea5e9` 블루

## 5. 애니메이션 목록

| 이름 | 대상 | 동작 |
|---|---|---|
| `drift` | 배경 블롭 | 8s 무한, translate + scale |
| `pulse-ring` | 프로필 사진 | 3s 무한, box-shadow 맥박 |
| `fadeUp` | 탭 콘텐츠 진입 | 0.35s, opacity 0→1 + translateY 18→0 |
| hover scale | 갤러리 카드 | scale(1.05) + translateY(-2px) |
| hover slide | 링크 아이템 | translateX(6px) |
| tab active | 탭 버튼 | scale(1.04) + 배경 전환 |

## 6. 기술 스택

- **단일 파일**: `profile.html`
- **외부 의존성**: Google Fonts CDN만 (오프라인 시 폴백 폰트 지정)
