=== Allsky AP 모드 설치 가이드 ===

[ 1단계: 이 파일(allsky-master-ap.zip)을 라즈베리파이로 전송 ]

방법 A - WinSCP 사용 (GUI, 추천):
  1. WinSCP 설치: https://winscp.net/
  2. 새 세션 > SFTP, 호스트: 192.168.32.1, 사용자: pi
  3. allsky-master-ap.zip 을 /home/pi/ 로 드래그&드롭

방법 B - scp 명령어 (PowerShell):
  scp "C:\Users\flyer\my-first-vibe\allsky-ap-mod\allsky-master-ap.zip" pi@192.168.32.1:/home/pi/

[ 2단계: 라즈베리파이에서 - AP 모드 일시 중지 후 인터넷 연결 ]

  sudo systemctl stop hostapd dnsmasq
  # 그 후 raspi-config 로 일반 WiFi 연결 (Network Options > Wireless LAN)
  sudo raspi-config

[ 3단계: 라즈베리파이에서 - allsky 설치 ]

  cd /home/pi
  unzip allsky-master-ap.zip
  cd allsky-master
  ./install.sh

  # 설치 중 카메라 타입 선택 시 RPi (라즈베리파이 공식 카메라) 선택

[ 4단계: 라즈베리파이에서 - AP 모드 복구 ]

  sudo systemctl start hostapd dnsmasq
  # 또는 재부팅: sudo reboot

[ 5단계: 접속 확인 ]

  PC/스마트폰 WiFi에서 AP 네트워크 재접속
  브라우저 주소창: http://192.168.32.1/allsky/

=== 수정된 파일 목록 (3개) ===

1. html/allsky/getForecast.php

   - 실패 시 친화적 메시지 반환 (UI 오류 없음)
